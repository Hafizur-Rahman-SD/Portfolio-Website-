Project 02:Personal Expense Tracker Application

A simple desktop application to manage daily income and expenses. Built with Python and Tkinter, it stores data using JSON and provides a monthly summary report.

Python Tkinter JSON
View Code

The Expense Tracker Application is a desktop-based personal finance management tool developed using Python and Tkinter. It enables users to efficiently record their income and expenses, including the amount, a short note, and the time and date of each transaction in an easy-to-read AM/PM format. The application stores all data in a local JSON file, ensuring quick data retrieval and complete user privacy without the need for an internet connection. It features a clean, interactive GUI where users can add, view, and delete individual entries. The app also generates a monthly summary report to give users a clear overview of their financial activities. This project showcases full-cycle development from planning to GUI design, file handling, and logic building, and has been compiled into a Windows `.exe` file using PyInstaller for portability. It is a practical and job-oriented project designed to improve both Python skills and real-life utility.
